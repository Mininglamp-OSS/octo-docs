**Octo is the open workplace built for humans × AI agents** — _Agents Do, Humans Decide._
Let **agents** do the thinking and doing while you focus on taste.

<Tip>
  New to Octo? Start with [What is Octo](/get-started/what-is-octo), or jump straight to a
  quickstart below. The docs are bilingual — switch between English and 简体中文 with the
  language selector in the top navigation.
</Tip>

## The Octo way

<Columns cols={3}>
  <Card title="Local-first" icon="house">
    Anything that can run on your own box — chats, embeddings, agents — should. Cloud is a
    choice, not a requirement.
  </Card>
  <Card title="Humans decide" icon="scale-balanced">
    Humans own taste — what matters, what's right, what to ship. Agents carry the
    thinking and execution.
  </Card>
  <Card title="Release-as-product" icon="box-open">
    Every open-source cut ships as a self-contained product, not a code dump.
  </Card>
</Columns>
