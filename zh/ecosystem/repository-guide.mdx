Octo 分布在 [Mininglamp-OSS](https://github.com/Mininglamp-OSS) 下的 **31 个仓库**中。
本门户按*你想做什么*来组织；本页则是按*仓库*绘制的地图，面向贡献者以及任何想理清方向的人。

## 各部分如何协同

```mermaid
graph TD
  subgraph Clients
    web[octo-web]
    android[octo-android]
    ios[octo-ios]
    ext[octo-chrome-extension]
    admin[octo-admin]
  end
  subgraph Core
    server[octo-server]
    fleet[octo-fleet]
    matter[octo-matter]
    summary[octo-smart-summary]
  end
  im[octo-im / WuKongIM]
  subgraph Shared
    lib[octo-lib]
    auth[octo-auth]
    adapters[octo-adapters]
  end
  web --> server
  admin --> server
  server --> im
  server --> matter
  server --> summary
  server --> fleet
  server -.uses.-> lib
  server -.uses.-> auth
  server --> adapters
```

## 核心服务（Go）

| 仓库 | 职责 |
|---|---|
| [`octo-server`](https://github.com/Mininglamp-OSS/octo-server) | 后端 API · 编排 · Lobster 调度 · 驱动 WuKongIM |
| [`octo-fleet`](https://github.com/Mininglamp-OSS/octo-fleet) | 运行时与 bot/守护进程编排（从 server 拆分而来） |
| [`octo-matter`](https://github.com/Mininglamp-OSS/octo-matter) | 任务 / 待办 / matter 微服务 |
| [`octo-smart-summary`](https://github.com/Mininglamp-OSS/octo-smart-summary) | LLM 会话摘要 |
| [`octo-mail`](https://github.com/Mininglamp-OSS/octo-mail) | 多租户邮件服务器内核（IMAP/JMAP/SMTP/REST） |
| [`octo-speech`](https://github.com/Mininglamp-OSS/octo-speech) | 语音转文字微服务 |
| [`octo-search-indexer`](https://github.com/Mininglamp-OSS/octo-search-indexer) | 消息搜索写入 + 索引层 |
| [`octo-message-export-api`](https://github.com/Mininglamp-OSS/octo-message-export-api) | 异步批量消息导出 |
| [`octo-version-sync`](https://github.com/Mininglamp-OSS/octo-version-sync) | 上游版本聚合器 → `version.json` |
| [`octo-marketplace`](https://github.com/Mininglamp-OSS/octo-marketplace) | 技能 / MCP 市场控制平面（脚手架） |
