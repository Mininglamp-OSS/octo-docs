### Send a message to a channel

`POST /v1/bot/sendMessage` · CLI 命令: `message.send` · 🟠 写

App Bot: DM-only (channel_type=1) and requires friend relationship. User Bot: DM + group + thread (requires membership).

**请求体**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `channel_id` | string | ✓ |  |
| `channel_type` | integer | ✓ | 1=DM, 2=group, 5=thread |
| `stream_no` | string |  | optional stream session identifier |
| `on_behalf_of` | string |  | OBO: send as this user persona; requires an active OBO grant + channel scope |
| `payload` | object | ✓ | message payload (content, type, mention, etc.). Pass via --data (object, not a flag). |

**响应**

| 状态 | 说明 |
|---|---|
| `200` | Send result |
