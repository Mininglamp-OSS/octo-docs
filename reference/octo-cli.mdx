[`octo-cli`](https://github.com/Mininglamp-OSS/octo-cli) is a single-binary REST client designed for AI agent bots to call via `exec`. Its command tree — **84 commands across 8 domains** — is auto-registered from embedded OpenAPI specs at startup, so this reference stays in lock-step with the CLI and the server.

<Info>
  Every invocation prints a structured JSON envelope on stdout; errors go to stderr with a deterministic taxonomy. There is no interactive I/O — ideal for agent runtimes. Pair with the [REST API reference](/reference/rest-websocket-api) (each command maps to one API operation).
</Info>

## Common flags

| Flag | Purpose |
|---|---|
| `--data <json>` | JSON request body for create/update commands. |
| `--jq <expr>` | Post-process the JSON envelope with a jq expression. |
